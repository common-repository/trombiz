== Description ==

Ce plugin vous permet d'afficher en widget les dernières photos de votre compte trombiz.com sur votre site/blog wordpress
Il faudra avant tout créer un compte sur le site trombiz.com : http://www.trombiz.com/p-register

== Installation ==

Cherchez le plugin via le moteur de recherche chez wordpress : http://wordpress.org/extend/plugins/search.php?q=trombiz&sort=
Cliquez dessus, et cherchez le bouton pour l'installer, il apparaitra dans votre interface Plugins.

1. Upload `trombiz.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
